# Pryes Brewing Company - Brand Guidelines

This repository contains the official brand guidelines and assets for Pryes Brewing Company.

## Overview

This project serves as a central hub for all brand-related materials, ensuring consistent brand representation across all touchpoints.

## Contents

- **logos/** - Official Pryes Brewing logos in various formats
- **colors/** - Brand color palettes and usage guidelines
- **typography/** - Approved fonts and type treatments
- **imagery/** - Photography and illustration guidelines
- **templates/** - Brand templates for common materials
- **guidelines/** - Detailed brand usage documentation

## Getting Started

Browse the directories above to find the assets and guidelines you need. Each section contains specific documentation on proper usage.
